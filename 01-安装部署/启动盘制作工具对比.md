## 1. Ventoy — 多系统启动神器

**最适合：IT运维、系统爱好者、需要经常装不同系统**

### 核心优势
- 安装一次，之后只需「复制/粘贴」ISO文件
- 一个U盘装 N 个系统（Windows/Linux/PE）
- U盘还能当普通存储用
- 支持 216+ 操作系统

### 使用方法
```
1. 下载 Ventoy → 解压运行 Ventoy2Disk.exe
2. 插入U盘 → 选择盘符 → 点击 Install
3. 把 ISO 文件直接复制到U盘（路径不能有中文/空格）
4. 启动时显示菜单，选择要安装的系统
```

### 避坑指南
- ⚠️ BIOS 开启安全启动会报错 `Security violation` → 关闭 Secure Boot
- ⚠️ 文件过多会导致启动慢 → 指定扫描目录
- ⚠️ ISO 文件路径不能包含中文或空格

### 官网
- https://www.ventoy.net

---

## 2. Rufus — 单系统最快选择

**最适合：小白用户、偶尔重装Windows**

### 核心优势
- 体积仅 5MB，无需安装
- 一键绕过 Win11 TPM/内存限制
- 写入速度极快

### 使用方法
```
1. 下载 Rufus.exe（免安装）
2. 选择U盘 → 选择ISO镜像 → 点击「开始」
3. 弹窗提示绕过Win11限制 → 勾选确认
4. 等待写入完成
```

### 官网
- https://rufus.ie

---

## 3. WinPE（微PE工具箱）— 系统维护工具包

**最适合：系统维护、数据恢复、磁盘分区**

### 核心优势
- 自带分区工具、数据恢复、密码重置等
- 不依赖原系统，可修复崩溃电脑
- 集成常用运维工具

### 使用方法
```
1. 下载微PE工具箱
2. 运行安装程序 → 选择「安装到U盘」
3. 启动后进入PE桌面，使用内置工具
```

### 与Ventoy配合
可以把微PE的ISO放入VentoyU盘，成为多系统之一。

### 官网
- https://www.wepe.com.cn

## 4. 快速选择指南

| 你的场景 | 推荐工具 |
|---------|---------|
| 小白，只装一次Windows | **Rufus** |
| 经常给人装各种系统 | **Ventoy** |
| 维修电脑、数据恢复 | **WinPE + Ventoy** |
| 玩树莓派/Linux发行版 | **BalenaEtcher** 或 Rufus |
| 老旧电脑（2010年前） | **UltraISO**（Legacy BIOS兼容好） |
