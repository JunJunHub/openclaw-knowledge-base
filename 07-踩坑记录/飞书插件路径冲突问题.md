# 问题

启动时报错：
```
Cannot find module '/home/junjun/.nvm/versions/node/v24.14.0/lib/node_modules/openclaw-cn/dist/plugin-sdk/index.js/feishu'
```

# 原因

OpenClaw 同时存在：
- **飞书信道**（`channels.feishu`）- 内置通道
- **飞书插件**（`plugins.entries.feishu`）- 外部插件

两者冲突，导致路径解析错误。

# 解决

在 `~/.openclaw/openclaw.json` 中禁用飞书插件入口：

```json
{
  "plugins": {
    "entries": {
      "feishu": {
        "enabled": false
      }
    }
  }
}
```

# 经验

- OpenClaw 的 `channels.feishu` 是内置功能，不需要插件
- 插件与信道同时启用会导致冲突
- 禁用插件入口后，飞书信道正常工作
