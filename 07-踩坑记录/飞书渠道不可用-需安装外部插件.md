# 飞书渠道不可用 - 需安装外部插件

> **分类**：踩坑记录
> **相关文档**：[[OpenClaw插件全览]]

## 问题现象

发送消息到飞书时报错：

```
❌ 发送失败: 飞书渠道不可用，需要安装官方外部插件
错误信息: Channel is unavailable: feishu
```

## 原因分析

飞书不在 OpenClaw 内置渠道列表中，需要安装官方外部插件。

## 解决方案

```bash
# 方法一：直接安装插件
openclaw plugins install @openclaw/feishu

# 方法二：使用 doctor 自动修复
openclaw doctor --fix
```

## 注意事项

- 飞书是外部插件，需要单独安装
- 安装后需要配置飞书应用凭证
- 可能需要重启 Gateway 服务

## 相关配置

安装后需要在 `openclaw.json` 中配置飞书渠道：

```json
{
  "channels": {
    "feishu": {
      "appId": "your_app_id",
      "appSecret": "your_app_secret"
    }
  }
}
```
