# 定时任务发送目标配置

## 查看所有定时任务

```bash
openclaw cron list
```

## 修改发送目标

```bash
# 修改为群组
openclaw cron update <task-id> --set 'delivery.to=chat:<群组ID>'

# 修改为私聊
openclaw cron update <task-id> --set 'delivery.to=user:<用户ID>'
```

## 群组 ID 获取

1. 在飞书群组中发送消息
2. 从消息元数据中获取 `chat_id`
3. 格式：`oc_` 开头的字符串
