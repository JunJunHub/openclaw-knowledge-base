# 目标

让 Agent 在飞书群组中自动回复所有消息，无需 @ 机器人。

# 配置方式

## 方法一：全局开放

```json
{
  "channels": {
    "feishu": {
      "groupPolicy": "open",
      "requireMention": false
    }
  }
}
```

所有群组自动回复，无需 @。

## 方法二：指定群组配置

```json
{
  "channels": {
    "feishu": {
      "groupPolicy": "allowlist",
      "groups": {
        "oc_xxx1": {
          "requireMention": false
        },
        "oc_xxx2": {
          "requireMention": false
        }
      }
    }
  }
}
```

仅指定群组自动回复。

# requireMention 参数

| 值 | 行为 |
|----|------|
| `true` | 必须机器人才能回复 |
| `false` | 所有消息都自动回复 |

# 注意事项

1. **群聊礼仪**：自动回复时，Agent 应遵守群聊规范，不刷屏
2. **权限配置**：需要 `im:message.group_msg` 权限
3. **重新发布**：修改权限后需要重新发布飞书应用

# 常见问题

## Q: 配置后群组仍无响应？

1. 检查飞书应用权限是否已发布
2. 确认 `groupPolicy` 设置正确
3. 查看群组 ID 是否匹配

## Q: 如何查看群组 ID？

让 MoltBot 在群里发消息，从 Gateway 日志中查看：

```bash
# 查看日志
tail -f ~/.openclaw/logs/gateway.log | grep chat_id
```
