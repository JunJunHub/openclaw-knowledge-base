# 背景

OpenClaw 支持多种搜索后端：
- Brave Search API
- Serper API (Google)
- 其他自定义搜索

# 推荐方案：Serper API

## 为什么选 Serper？

| 维度 | Serper | Brave Search |
|------|--------|--------------|
| 国内访问 | ✅ 可直接访问 | ❌ 需要代理 |
| 免费额度 | 2,500 次/月 | 2,000 次/月 |
| 搜索质量 | Google 结果 | Brave 自有 |
| 注册难度 | 低 | 中 |

## 配置方式

### 1. 获取 API Key

访问 https://serper.dev 注册账号，获取 API Key。

### 2. 配置 OpenClaw

编辑 `~/.openclaw/.env`：

```bash
SERPER_API_KEY=你的API Key
```

或在 `openclaw.json` 中：

```json
{
  "plugins": {
    "entries": {
      "serper": {
        "enabled": true
      }
    }
  }
}
```

### 3. 使用工具

- `serper_search` - 网页搜索
- `serper_scholar` - 学术搜索

# Agent 规范

**所有 Agent 必须优先使用 `serper_search`**

```
❌ 不要要求配置 Brave API Key
✅ 直接使用 serper_search 或 web_search
```
