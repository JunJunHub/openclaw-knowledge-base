# 搜索配置统一规范

## 背景

OpenClaw 默认使用 Brave Search API（`web_search` 工具），但在国内环境下存在以下问题：

1. **网络限制**：Brave Search API 需要代理访问
2. **API Key 申请**：需要额外注册 Brave 账号
3. **额度限制**：免费额度较少

## 解决方案：Serper API

### 为什么选择 Serper

| 对比项 | Brave Search | Serper API |
|--------|-------------|------------|
| 国内访问 | ❌ 需要代理 | ✅ 可直接访问 |
| 免费额度 | 2,000 次/月 | **2,500 次/月** |
| API Key 申请 | 需注册 Brave 账号 | 一键注册 |
| 搜索质量 | 自有搜索引擎 | Google 搜索结果 |
| 学术搜索 | ❌ 不支持 | ✅ `serper_scholar` |

### 配置步骤

#### 1. 获取 API Key

1. 访问 https://serper.dev
2. 使用 Google 账号一键登录
3. 在 Dashboard 复制 API Key

#### 2. 配置 OpenClaw

在 `~/.openclaw/.env` 中添加：

```bash
SERPER_API_KEY=your_api_key_here
```

#### 3. 安装 Serper 插件

```bash
cd ~/.openclaw/extensions
git clone https://github.com/openclaw/serper-plugin.git serper
```

#### 4. 启用插件

在 `openclaw.json` 中添加：

```json
{
  "plugins": {
    "serper": {
      "enabled": true
    }
  }
}
```

#### 5. 重启 Gateway

```bash
openclaw gateway restart
```

## 工具说明

### serper_search

网页搜索工具，返回 Google 搜索结果。

**参数**：
- `query` (必填): 搜索关键词
- `num` (可选): 返回结果数量，默认 5，最大 20
- `gl` (可选): 国家代码，默认 `cn`
- `hl` (可选): 语言代码，默认 `zh-CN`

**示例**：
```json
{
  "query": "OpenClaw 教程",
  "num": 10,
  "gl": "cn",
  "hl": "zh-CN"
}
```

### serper_scholar

学术搜索工具，搜索学术论文和文献。

**参数**：
- `query` (必填): 搜索关键词
- `num` (可选): 返回结果数量，默认 10，最大 20

**示例**：
```json
{
  "query": "LLM fine-tuning methods",
  "num": 10
}
```

## 使用规范

### ✅ 推荐做法

1. **统一使用 `serper_search`**：所有 Agent 默认使用 Serper
2. **学术搜索用 `serper_scholar`**：论文、研究类查询
3. **优先中文**：`gl: "cn"`, `hl: "zh-CN"` 获取中文结果
4. **控制数量**：日常搜索 5-10 条即可，减少 API 消耗

### ❌ 避免事项

1. **不要要求 Brave API Key**
2. **不要使用 `web_search` 工具**（国内访问受限）
3. **不要频繁调用**：合理控制搜索频率

## 成本估算

- 免费额度：2,500 次/月
- 超额费用：$2 / 1,000 次
- 月均使用：~500 次（日常使用足够）
- **预估成本：免费**

## 故障排查

### 问题 1：API Key 无效

**现象**：返回 `401 Unauthorized`

**解决**：
1. 检查 `.env` 中的 `SERPER_API_KEY` 是否正确
2. 确认 Key 没有过期（登录 serper.dev 查看）

### 问题 2：搜索无结果

**现象**：返回空数组

**解决**：
1. 检查网络连接
2. 尝试更换 `gl` 和 `hl` 参数
3. 简化搜索关键词

### 问题 3：插件未加载

**现象**：工具列表中没有 `serper_search`

**解决**：
1. 确认插件目录存在：`ls ~/.openclaw/extensions/serper`
2. 检查 `openclaw.json` 中插件已启用
3. 重启 gateway：`openclaw gateway restart`
