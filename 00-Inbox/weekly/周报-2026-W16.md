## 🛠️ 开发环境优化

### Python环境管理：从pyenv迁移到uv
- **决策**：弃用pyenv，全面拥抱uv（Rust编写，极速）
- **版本建议**：推荐Python 3.11/3.12，避免3.14（兼容性风险）
- **两种工作流**：
  - 项目开发模式：`uv python pin 3.12` → `uv venv` → `uv add requests`
  - 脚本工具模式：`uv python pin --global 3.12`
- **笔记**：`03-AI编程工具/Python环境管理-uv工作流.md`

### Samba文件共享故障排查
- **核心发现**：Samba"双重身份"机制（Linux用户 ≠ Samba用户）
- **标准排查清单**：
  1. `sudo pdbedit -L` 检查Samba用户
  2. Windows安全策略：启用"不安全的来宾登录"
  3. 文件夹权限：`sudo chown -R user:user /path`
  4. 重启服务：`sudo systemctl restart smbd`
- **一键恢复脚本更新**：`12-file-sharing.sh` 添加Samba账户创建提示
- **笔记**：`05-配置指南/Samba文件共享故障排查.md`
